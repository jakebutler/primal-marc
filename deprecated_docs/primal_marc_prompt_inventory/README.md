# Primal Marc — Prompt Inventory (Final)

This directory contains the complete, production-ready prompt inventory for **Primal Marc**,
optimized for a hackathon build and demo.

## Model Usage
- **All text generation:** MiniMax **M2**
- **Image generation:** MiniMax Image API
- **Video generation:** MiniMax / Hailuo 2.3 Fast

## Agents & Prompts
1. Idea → Brief Agent (`idea_brief_agent.md`)
2. Prompt Composer Agent (`prompt_composer_agent.md`)
3. Blog Generator Agent (`blog_generator_agent.md`)
4. Social Post Generator Agent (`social_post_agent.md`)
5. Image Prompt Generator (`image_prompt_generator.md`)

## Architecture Notes
- No orchestration agent
- Explicit procedural flow
- Prompts are versionable, inspectable, and demo-friendly

This inventory is intended to be exposed in submission materials.
